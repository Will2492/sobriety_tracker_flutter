# SobriOK Milestones & Telegram Integration

This document defines milestone recognition and optional Telegram-based messages in SobriOK.

---

## Milestone principles

- Milestones mark time, not achievement
- Milestones are optional and non-judgmental
- Milestones are never blocked by Premium
- Language is calm and pressure-free
- No verification or claims of sobriety

---

## Milestone schedule

- 90 days
- 180 days
- 270 days
- 365 days

Milestones are calculated based on the user-defined start date.

---

## In-app milestone experience

Each milestone includes:
- A dedicated milestone screen
- A short reflective message
- Optional action: "Receive a message in Telegram"

The user may skip or ignore milestones without consequences.

---

## Telegram integration overview

Telegram integration is optional and external to the app.

### Flow
1. User reaches a milestone
2. App generates a one-time token
3. User opens SobriOK Telegram bot
4. User submits the token
5. Bot sends a milestone message

No personal data is shared publicly.

## Premium relation

- Basic milestone messages are free
- Premium may include:
  - Extended reflections
  - Visual summaries
  - Archive of milestone history
- Premium never unlocks milestone access

---

## Compliance & safety

- No rewards tied to behavior
- No promises or validation claims
- No medical language
- Telegram is optional and external
