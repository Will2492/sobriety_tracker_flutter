# SobriOK

SobriOK is a calm, ad-free sobriety tracking app focused on awareness, consistency, and honest self-reflection.

The app is designed as a supportive companion â€” not a medical tool and not a promise of results.

---

## Core principles

- No pressure, no shame
- The core experience is always free
- Optional premium features enhance insight, not access
- Privacy-first approach
- Calm, neutral tone

---

## Features (planned)

### Free
- Sobriety tracker (days / hours / minutes)
- Start date storage
- Relapse / reset flow
- Milestones every 90 days (90 / 180 / 270 / 365)
- Local data storage
- Language selection (English / Russian)

### Premium (optional subscription)
- Advanced analytics and patterns
- Money and time saved calculations
- Daily intentions and reflections
- Cloud backup
- Enhanced milestone summaries
- Additional Telegram-based bonuses
